# Getting Started With Erpnext

Hay muchas manera de comenzar a utilizar EPAAS.

### 1\. Ver el Demo

Si deseas entrar en contacto con la interfaz de usuario de EPAAS y **sentir** la aplicación, solo tienes que ver el demo en:
see the demo at:

  * <https://demo.epaas.com>

### 2\. Comienza con una cuenta gratis en EPAAS.com


EPAAS.com es manejado por la organización (Frappé) que publicó EPAAS.
Puedes iniciar con su propia cuenta en [registrandote en la página](https://epaas.com).

También, puedes hostear tu aplicación en epaas.com comprando un plan de alojamiento.
	De esta forma, estas aportando a la organización que desarrolla y mejora EPAAS.
  También obten soporte de uno-a-uno con los planes de alojamiento.

### 3\. Descarga una Maquina Virtual

Para evitar las molestias de instalar el sistema, EPAAS está disponible como una image virtual (un sistema operativo completo con EPAAS instalado).
Puedes usarla en **cualquier** plataforma incluyendo Microsoft Windows.

[Click aquí para ver las instrucciones de como usar la imagen](https://epaas.com/download)

### 4\. Instalar EPAAS en su ordenador Unix/Linux/Mac

En caso de estar relacionado con la instalación de aplicaciones en plataformas *nix, leer las instrucciones de como instalarlo usando [Frappé Bench](https://github.com/dataent/bench).
