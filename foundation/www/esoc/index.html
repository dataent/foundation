EPAAS Summer of Code

EPAAS Summer of Code is an initiative by the EPAAS Foundation, in similar lines to Google Summer of Code (GSoC) to help bring more young and budding developers into the world of Open Source Software development. Students will spend 3 months of their summer time working with EPAAS and contributing to open source development, all while earning a stipend!
